# Revision Problems Email Notification
Do you have list of leetcode problems to be revised and unable to track them? Then use this automated script which generates email based on your revision problem list.

Follow these steps to setup email notification

1. Download the google sheet that is present in the repository. Ensure you modify the content and not the columns as column orders are used in automation script to generate emails.


2. Open the google sheet in google sheets (https://www.google.com/sheets/about/)


3. Upon opening the google sheet, select Extensions menu and choose Apps Script option.

<img width="1431" alt="Screen Shot 2023-06-02 at 9 53 19 PM" src="https://github.com/nithin1shastry/revisionEmailNotification/assets/30688704/ee6ca0ce-8aef-4254-8296-e7d8b3054d68">


4. Select Editor section. Copy paste Code.gs file code into Code.gs file section. You can rename the title accordingly.

<img width="1276" alt="Screen Shot 2023-06-02 at 9 38 54 PM" src="https://github.com/nithin1shastry/revisionEmailNotification/assets/30688704/63e4249e-ec2b-42f3-8f0c-3aaafd3558ff">


5. Select Trigger section, Click on Add trigger button.
